# Student Placement Predictor
## Project Overview
This project predicts whether a student is likely to be placed or not based on various input features like academic and skill realted information using machine learning.
The application is built using Python ,Flask,HTML,CSS.

## Features
1.Predicts placement status
2.Easy to use interface
3.machine learning model integration
4.Responsive design

## Technologies Used
-Python
-Flask
-Numpy
-Pandas
-Scikit-learn
-seaborn
-HTML
-CSS

## Dataset Features
-IQ
-Previous Semester Result
-CGPA
-Academic Performance
-Internship Experience
-Communication Skills
-Extra curricular score
-Projects Completed

## Machine Learning Algorithm
Selected the 'Random Forest Classifier' ML algorithm as it was showing the better accuracy than rest of the models.

## How to run
1.Install python
2.Install flask
3.Install required libraries (numpy,scikit-lean,pandas,matplot.lib,seaborn)
4.Run
python app.py

http://127.0.0:5000/
